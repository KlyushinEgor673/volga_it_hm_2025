Приложение для AuroraOS на Flutter — просмотрщик фотографий с расширенными функциями. Приложение отображает все изображения, хранящиеся на устройстве, извлекает из них метаданные и показывает места съемки на карте. Также предоставляет инструменты для редактирования: обрезку фотографий и изменение их названий.
